# divisor-de-letra
Projeto de um divisor de letras para usarmos na projeção da igreja

# Rodar o projeto
Necessário ter o node instalado na versão 18 ou superior.
Para rodar o projeto, use o comando:

```bash
npm run dev
```

# Testes

Necessário ter o node instalado na versão 18 ou superior.
Para rodar os testes, use o comando:

```bash
npm run test
```
